# دراسة جدوى — تأسيس بيت خبرة تقني

موقع تفاعلي يعرض دراسة جدوى لتأسيس بيت خبرة تقني في المملكة العربية السعودية.

## المحتوى

- `index.html` — الموقع الكامل (صفحة واحدة مستقلة، تتضمن التنسيق والبرمجة).

## التشغيل

افتح `index.html` مباشرة في المتصفح، أو شغّل خادمًا محليًا:

```bash
python3 -m http.server 8000
```

ثم افتح <http://localhost:8000>.

## النشر

الموقع صفحة HTML مستقلة بالكامل (RTL، يدعم العربية، الوضع الفاتح/الداكن)، ويمكن نشرها عبر أي استضافة ثابتة مثل GitHub Pages.

# Feasibility Study — Technical Expertise House

Interactive single-page website presenting a feasibility study for establishing a technical expertise house in Saudi Arabia.

- `index.html` — the complete, self-contained website (styles and scripts inlined).

It is a fully standalone, RTL Arabic page with light/dark mode and can be deployed to any static host such as GitHub Pages.
